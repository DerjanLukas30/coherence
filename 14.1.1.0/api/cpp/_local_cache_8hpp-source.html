00359         /**
00360         * Get all the specified keys, if they are in the cache. For each key
00361         * that is in the cache, that key and its corresponding value will be
00362         * placed in the map that is returned by this method. The absence of
00363         * a key in the returned map indicates that it was not in the cache,
00364         * which may imply (for caches that can load behind the scenes) that
00365         * the requested data could not be loaded.

00367         * The result of this method is defined to be semantically the same as
00368         * the following implementation, without regards to threading issues:
00369         *
00370         * <pre><tt>
00371         * Map map = new AnyMap(); // could be hash map or ...
00372         * for (Iterator iter = col.iterator(); iter.hasNext(); )
00373         *     {
00374         *     Object oKey = iter.next();
00375         *     Object oVal = get(oKey);
00376         *     if (oVal != null || containsKey(oKey))
00377         *         {
00378         *         map.put(oKey, oVal);
00379         *         }
00380         *     }
00381         * return map;
