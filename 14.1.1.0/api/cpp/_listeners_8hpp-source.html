00017 /**
00018 * Provides a simple, efficient, and thread-safe implementation of a list
00019 * of event listeners.
00020 *
00021 * The implementation is optimized based on the assumption that listeners are
00022 * added and removed relatively rarely, and that the list of listeners is
00023 * requested relatively often.
00024 *
00025 * Thread safety is implemented by synchronizing on all methods that modify
00026 * any data member of the class.  Read-only methods are not synchronized.
00027 *
00028 * @author jh  2007.12.13
00029 */
