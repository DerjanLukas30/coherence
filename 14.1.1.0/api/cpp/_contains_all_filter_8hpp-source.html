00020 /**
00021 * Filter which tests a Collection or ObjectArray value returned from
00022 * a method invocation for containment of all values in a Set.
00023 *
00024 * More formally, if the specified extractor returns a Collection,
00025 * #evaluate is functionally equivalent to the following
00026 * code:
00027 * <pre>
00028 * return ((Collection) extract(o))->containsAll((Collection) getValue());
00029 * </pre>
00030 * If the specified method returns an ObjectArray, #evaluate is functionally
00031 * equivalent to the following code:
00032 * <pre>
00033 * return Collections->asList((ObjectArray::View) extract(o))->containsAll(
00034 *        (Collection) getValue());
00035 * </pre>
00036 * @author djl  2008.04.07
00037 */
