00024 /**
00025 * The UserTypeWriter implementation is a contextually-aware PofWriter whose
00026 * purpose is to write the properties of a value of a specified user type. The
00027 * "contextual awareness" refers to the fact that the UserTypeWriter maintains
00028 * state about the type identifier, the PofWriter's property index position
00029 * within the user type value, and a PofContext that may differ from the
00030 * PofContext that provided the PofSerializer which is using this
00031 * UserTypeWriter to serialize a user type.
00032 *
00033 * @author jh  2008.04.11
00034 */
00035 class COH_EXPORT UserTypeWriter
