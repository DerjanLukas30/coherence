Interface IStatus

An IStatus represents an asynchronous IRequest sent to a peer.

Namespace: Tangosol.Net.Messaging
Assembly: Coherence.Core.dll
Syntax
public interface IStatus
Remarks

The status of the IRequest can be determined by checking the value of the IsClosed property. If it returns false, the request is still in progress. A return value of true indicates that the request has either completed successfully, completed unsuccessfully, or has been canceled.

When the IRequest completes, the IResponse sent by the peer can be retrieved from the Response property. If this property value is null, the IRequest was explicitly canceled (Cancel()).

Rather than constantly polling the IRequest for the outcome of the IRequest, a thread can alternatively use the IStatus to WaitForResponse(Int64) wait for the IRequest to complete.
