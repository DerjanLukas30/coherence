AbsolutePath

Gets an absolute path of this resource.

Declaration
public string AbsolutePath { get; set; }
Property Value
Type Description
System.String
Remarks

The exact value returned will depend on the specific resource implementation.

For example, file system-based resources that support relative path resolution, such as FileResource and , should return an absolute path of the resource on the file system. Other resource types should return fully qualified URI for the resource, which is basically the value of the Uri property.
