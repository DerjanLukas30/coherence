Interface IResource

Interface for access to different types of resources that can provide a System.IO.Stream to read from.

Namespace: Tangosol.IO.Resources
Assembly: Coherence.Core.dll
Syntax
public interface IResource
Remarks

This interface encapsulates a resource descriptor that abstracts away from the underlying type of resource; possible resource types include files, URLs, web resources and assmbly embedded resources.

AbsolutePath

Gets an absolute path of this resource.

Declaration
string AbsolutePath { get; }
Property Value
Type Description
System.String

The absolute path of this resource.
Remarks

The exact value returned will depend on the specific resource implementation.

For example, file system-based resources that support relative path resolution, such as FileResource and , should return an absolute path of the resource on the file system. Other resource types should return fully qualified URI for the resource, which is basically the value of the Uri property.

caution

Clients of this interface must be aware that every call of this method will create a fresh System.IO.Stream; it is the responsibility of the calling code to close any such System.IO.Stream.
