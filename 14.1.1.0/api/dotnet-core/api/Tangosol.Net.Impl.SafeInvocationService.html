Parameters
Type Name Description
System.Boolean drain

If true and the wrapped IInvocationService is restarted, the calling thread will be blocked until the wrapped IInvocationService event dispatcher queue is empty and all outstanding tasks have been executed.
Returns
Type Description
IInvocationService

The running wrapped IInvocationService.
Remarks

This method ensures that the returned IInvocationService is running before returning it. If the IInvocationService is not running and has not been explicitly stopped, the IInvocationService is restarted.

The IInvocable object to distribute to the specified members in order to be invoked on those members.
System.Collections.ICollection col

Optional collection of cluster members to which the IInvocable object will be distributed; if null, the IInvocable object will be distributed to all cluster members that are running this service.
Returns
Type Description
System.Collections.IDictionary

An IDictionary of result objects keyed by IMember object.
Remarks

This method will not return until the specified members have completed their processing, failed in their processing, or died trying.

IMembers that are specified but are not currently running the IInvocationService will not invoke the specified IInvocable object.

IMembers that leave (gracefully or otherwise) before the invocation completes will not register a result, and the amount of processing that completed is indeterminate.
