DestroyCacheRequest

The DestroyCacheRequest is a CacheServiceRequest sent to destroy a NamedCache running within a Coherence cluster.

EnsureCacheRequest

The EnsureCacheRequest is a CacheServiceRequest sent to obtain a remote reference (i.e. a Channel) to a NamedCache running within a Coherence cluster.
