Interface IChannel

An IChannel is a communication construct that allows one or more threads to send and receive IMessage objects via a IConnection.

Namespace: Tangosol.Net.Messaging
Assembly: Coherence.Core.dll
Syntax
public interface IChannel
Remarks

IChannel objects are created from an IConnection. Once created, an IChannel can be used to:

  • asynchronously Send(IMessage) send an IMessage
  • asynchronously Send(IRequest) send an IRequest
  • synchronously Request(IRequest) send an IRequest
  • asynchronously Receiver receive an IMessage

Once an IChannel has been closed, any attempt to send a IMessage using the IChannel may result in an exception.

All IChannel implementations must be fully thread-safe.

Close()

Close the IChannel and reclaim all resources held by the IChannel.

Declaration
void Close()
Remarks

When this method is invoked, it will not return until IMessage processing has been shut down in an orderly fashion. This means that the IReceiver object associated with this IChannel (if any) have finished processing and that all pending requests are completed or canceled. If the IReceiver is processing an IMessage at the time when close is invoked, all the facilities of the IChannel must remain available until it finishes.

If the IChannel is not open, calling this method has no effect.
