Interface IResponse

IResponse is the root interface for all response messages sent by peer endpoints through an IChannel.

Syntax
public interface IResponse : IMessage, IRunnable
Remarks

IResponse objects are created by an IMessageFactory and are associated with a corresponding IRequest. The status of the IResponse may be determined by reading the IsFailure property value:

  • if false, the IRequest was successfully processed by the receiver and the IResponse object contains a valid result
  • if true, an exception occured while processing the IRequest

IsFailure

Determine if an exception occured while processing the IRequest.

Declaration
bool IsFailure { get; set; }
Property Value
Type Description
System.Boolean

false if the IRequest was processed successfully; true if an exception occured while processing the IRequest.
Remarks

If this property value is false, the result of processing the IRequest can be determined by reading the Result property value.

If this property value is true, Result may return the cause of the failure (in the form of an Exception object).
