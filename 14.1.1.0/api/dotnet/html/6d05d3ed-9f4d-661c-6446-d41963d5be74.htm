TransformationState describes how a CacheEvent has been or should be transformed.

public enum TransformationState
Members
  Member nameValueDescription
NON_TRANSFORMABLE0 Value used to indicate that an event is non-transformable and should not be passed to any transformer-based listeners.
TRANSFORMABLE1 Value used to indicate that an event is transformable and could be passed to transformer-based listeners.
TRANSFORMED2 Value used to indicate that an event has been transformed, and should only be passed to transformer-based listeners.
