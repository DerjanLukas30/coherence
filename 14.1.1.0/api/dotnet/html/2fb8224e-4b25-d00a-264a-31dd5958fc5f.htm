Remarks
An implementation of IPofSerializer is required to follow the following steps in sequence for writing out an object of a user type:
  1. The implementation may write any combination of the properties of the user type by using the "write" methods of the IPofWriter, but it must do so in the order of the property indexes.
  2. After all desired properties of the user type have been written, the implementation must terminate the writing of the user type by calling WriteRemainder(Binary).
